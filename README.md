# avito-feed

Автоматически публикуемый фид Авито-автозагрузки.

- `avito-feed.xlsx` — актуальный фид (цены целые, ceil).
- Обновляется автоматически каждые 4 часа.
- Прямая ссылка для автозагрузки:
  `https://raw.githubusercontent.com/forshulman-cloud/avito-feed/main/avito-feed.xlsx`

Временное решение. Содержит только фид — никаких скриптов и секретов.
